\section*{Overview}

This document contains general information for the ``Hands-on Introduction to R'' lectures.

\section{Access to the \texttt{R} language interpreter and the \texttt{RStudio} IDE}

The lectures require access to an \texttt{R} language interpreter and the \texttt{RStudio} IDE.
Below, you will find a few options:

\begin{enumerate}
  \item If you have a valid CHPC account you can use CHPC's
        \href{http://ondemand.chpc.utah.edu/}{Ondemand Web Portal}.
        After logging into OnDemand you can launch the \texttt{RStudioServer} application,
        which contains \texttt{R} and \texttt{RStudio}.

  \item You can sign up for an account on
        \href{https://posit.cloud/}{posit Cloud}
        (which provides 25~hours/month of free computing time).
        posit~Cloud contains \texttt{R} and \texttt{RStudio}.

  \item You can also install \texttt{R} and \texttt{RStudio} on your machine.
        The \texttt{R} command-line interpreter can be downloaded from the
        \href{https://cran.r-project.org/}{Comprehensive R Archive Network (CRAN)}.
        The \texttt{RStudio} Desktop IDE can be downloaded from the
        \href{https://posit.co/downloads/}{posit downloads page}.
\end{enumerate}

\section{Obtaining the lecture material}

All the course material can be obtained from:
\href{https://github.com/wcardoen/IntroToR}{https://github.com/wcardoen/IntroToR}.

If you have \texttt{git} installed, you can obtain the material as follows:
\begin{verbatim}
git clone https://github.com/wcardoen/IntroToR.git
\end{verbatim}

Another option is to download the zip file at:\\
\href{https://github.com/wcardoen/IntroToR/archive/refs/heads/main.zip}{%
  https://github.com/wcardoen/IntroToR/archive/refs/heads/main.zip}
